YourKids Redesign — Developer Brief

Direction: "Warm & Playful" (design 1a) · 3 July 2026 · Reference build: demo/yourkids-1a-demo.html (standalone HTML + CSS with all tokens as CSS variables — lift the stylesheet directly)

1 · Goals — what this redesign fixes

  • Clear CTA hierarchy. One primary action (coral "Get the weekly email" in the header, repeated as the newsletter band), one secondary pair in the hero (teal "Browse by topic", outline "Latest articles"). Currently the homepage has no visual priority between links.
  • No duplicated content. The current homepage shows the same three articles in both "Latest" and "A Bit of Good News". New structure: one "Featured this week" row (3 cards, deduplicated), then topics, then newsletter. If a Good News section returns later, it must exclude articles already shown above.
  • Working newsletter signup. Replace both "coming soon" notes with a real email capture (hero header button + full-width teal band, POST /subscribe). This is the site's main retention loop — treat as launch-blocking.
  • Visible search. Search moves from a nav link to a persistent field in the header.
  • Active nav state. Pill nav with the current section highlighted (sand background), hover feedback on all links/cards.

4 · Shape, elevation, layout

  • Radii: pills/buttons/chips/search 999px · cards 20px · card images 14px · topic tiles 18px · hero art & newsletter band 24px. Rounded everywhere — no square corners.
  • Shadows: cards 0 2px 8px rgba(90,70,40,.07); hover 0 6px 18px rgba(90,70,40,.13) + translateY(-3px). Warm-toned shadow, never pure black.
  • Page: content max-width 1120px, centered; horizontal gutter 40px; sections separated by 48px bottom padding.
  • Grids: hero 1fr / 420px, gap 48px, vertically centered · article cards repeat(3, 1fr), gap 20px · topic tiles repeat(6, 1fr), gap 14px. All spacing via flex/grid gap, not margins.

5 · Page structure & components (top to bottom)

  1. Header — flex row, 18px vertical padding, 1px bottom border. Order: logo · pill nav (Home, Baby & Toddler, Topics, Articles, Curated Links, Recommended) · right-aligned search field (1.5px #E4D9C4 border) · coral primary button "Get the weekly email" (anchors to newsletter band). "Baby & Toddler" keeps its dropdown (Baby week by week, Feeding, Baby health, Toddler, Open the app); Editorials, Deals, Trusted Sites move to the footer.
  2. Hero — left column: teal-tint badge "Updated every week", H1 (two lines), standfirst (max-width 480px), CTA pair (teal solid + outline). Right column: illustration on 24px-radius mat, height 300px.
  3. Featured this week — section head (H2 + right-aligned coral "All articles →"), then 3 article cards: image on tinted mat (140px, rotate mats coral-tint / teal-tint / sand) → tag chips → Baloo title → 2-line excerpt. Max 3 cards; no duplicates with any other section.
  4. Browse by topic — 6 tiles (name + article count), tint rotation coral → teal → sand. Whole tile is the link.
  5. Newsletter band — full-width teal-deep rounded band: heading "A short, friendly email every Sunday", reassurance line ("No judgement, no spam, unsubscribe anytime"), inline email input (pill, white) + coral "Sign up" button (as in the mock — coral pops against the teal-deep band).
  6. Footer — keep current link structure (Explore column, digest note, © line), restyled on cream with sand hairlines. Not in the demo; follow header/nav styling.

6 · Interaction states

  • Nav pills: hover + active = sand background; active also darkens text to #3A342C.
  • Buttons: coral → #C55A38, teal → #2E8A7C on hover; outline button darkens border. Transitions ~150ms.
  • Cards & tiles: hover lift (translateY −3px / −2px) with the stronger shadow; entire card is one link target.
  • Focus: :focus-visible 3px teal outline, 2px offset, on every interactive element.

7 · Responsive & accessibility

  • ≤1000px: hero stacks (illustration above copy, 220px), cards 2-up, tiles 3-up, newsletter stacks centered, nav collapses to a hamburger drawer (drawer not in demo — reuse pill styling vertically).
  • ≤640px: cards 1-up, tiles 2-up. Touch targets ≥44px.
  • Contrast: all text/tint pairs in §3 pass WCAG AA at their sizes; re-check if you adjust tints. Chips are supplementary — never the only category indicator on article pages.
  • Semantic HTML as in the demo: nav, article, real form for signup, aria-current="page" on active nav item.
